# Platform Promise

The Platform Promise is the set of guarantees PlayOS makes to players
and developers. It is the contract that defines what the platform is
and what it will always be.

## For players

| Promise | What it means |
|---|---|
| **Your games are yours** | Purchased games are entitlements, not rentals. They work on any device you own. |
| **Your saves are yours** | Save data is local by default, optionally synced to the cloud. You can back it up, move it, delete it. |
| **Your device is yours** | Developer Mode is available. You can side-load, mod, and tinker. No remote kill switch. |
| **No always-online DRM** | Single-player games do not require an internet connection after install. |
| **One account, any store** | One player identity works across official, community, and OEM stores. |
| **Privacy by default** | Telemetry is opt-in. Your play data is not sold. See [Privacy Principles](../13-security-privacy-and-trust/08-privacy-principles.md). |

## For developers

| Promise | What it means |
|---|---|
| **Write once, play anywhere** | One binary runs on every PlayOS device. The Platform API abstracts the hardware. |
| **No royalties required** | The Platform API, Runtime, and Tools are free. Stores may charge their own fees. |
| **Specification-first** | The platform is defined by an open specification. Implementations follow the spec, not the other way around. |
| **Engine freedom** | Use Raylib, SDL, Godot, or a custom engine. The Platform API does not care. |
| **Long-term compatibility** | Binaries compiled today will run on PlayOS devices years from now. |
| **Self-publish or use a store** | Upload to the official marketplace, your own store, or both. |

## For OEMs and device makers

| Promise | What it means |
|---|---|
| **Build your own device** | Implement the Platform API and Runtime, pass certification, and you have a PlayOS device. |
| **Run your own store** | The marketplace is self-hostable. OEMs can bundle their own store. |
| **No per-device license fee** | The platform software is open source. Certification has a fee to cover testing costs. |
